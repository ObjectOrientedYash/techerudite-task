# Product Inventory System

Full-stack Product Inventory built with React, Node.js, GraphQL, PostgreSQL, Sequelize, and Zod.

## Features

- Add product (unique name)
- Multi-select categories
- Paginated listing
- Search by name
- Category dropdown filter (OR logic)
- Delete product
- Client + server validation

---

## Setup

### Backend

```bash
cd backend && npm install && npm run seed && npm start
```

### Frontend

```bash
cd frontend && npm install && npm run dev

```
